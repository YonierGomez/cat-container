Features

🏗️

Multi-Architecture

Runs on x86_64, ARM64 and ARMv7. Built with QEMU + Buildx on GitHub's free runners.

🔄

Auto-Release

Weekly check for new nginx versions. Automatic build, push and GitHub Release with upstream changelog.

🪶

Lightweight

Based on nginx:alpine. The entire image is around 7MB compressed.

✅

CI/CD Tested

Every PR runs a build + health check. No broken images reach Docker Hub.
